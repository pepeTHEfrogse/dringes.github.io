# DRINGES-OS
это моя операционная система котрую я делаю в одиночку.
Вот ее небольшое описание:
Он выделяется встроенным локальным ИИ-помощником Dring,
различные пространства, модульность на лету, кросс-совместимость, внутрисистемный DRINCoin,
режим ДНШ (доверяй, но шифруй),
а также Game & Create Hub (GCH).

Всего я планирую пять версий:
 
 1.DRINGES — базовая версия.

 2.DRINGES-eco — самая легкая из версий с урезанным функционалом, за использование будут начисляться DRINcoin'ы.
 
 3.DRINGES-ultra — самая тяжелая из версий, функционал будет мощным, что то вроде гибрида всех версий.
 
 4.DRINGES-gaming — ориентированная на игры, не будет подходить для творчества и тп.

 5.DRINGES-private — самая приватная версия, которая обеспечит безопасный веб-серфинг, но не убережет от ошибок пользователя.
