# Спецификация DRINCoin

##  Эмиссия
- Начальный запас: 1 000 DRIN
- Начисление: за полезные действия
- Сжигание: при трате в магазине

##  Способы заработка

| Действие | Награда | Лимит |
|---------|--------|------|
| Eco-режим (1 час) | 10 DRIN | 100/день |
| Найденный баг | 500 DRIN | 5/день |
| Создание контента | 1000 DRIN | 2/неделя |
| Тестирование | 300 DRIN | 10/неделя |
| Приглашение друга | 200 DRIN | нет |
